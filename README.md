# my-kde-plasma-theme
My personal KDE Plasma theme, combining components from multiple existing themes, patching them to fix visual issues, and manually editing some elements for a consistent style.
# Features
* Layan style
* Catppuccin Mocha colors
* Breeze-style window buttons
* Compatible with Burn-My-Windows and Wobbly Windows
